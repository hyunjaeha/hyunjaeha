## 소개

| 분야 | 설명 |
|---|---|
| UX 설계 | 사용자 흐름과 문제를 파악하고 개선합니다 |
| UI 디자인 | 직관적이고 일관된 화면을 설계합니다 |
| 웹 디자인 | 다양한 디바이스에 맞는 반응형 화면을 제작합니다 |
| 협업 | Figma와 웹 구조 이해를 바탕으로 소통합니다 |
