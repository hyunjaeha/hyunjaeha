## What I Do

### UI/UX Design
사용자 리서치, 정보 구조 설계, 와이어프레임, 프로토타입 제작을 통해  
서비스의 사용성과 흐름을 개선합니다.

### Web Design
브랜드의 분위기와 목적에 맞는 웹 화면을 설계하고,  
데스크탑, 랩탑, 테블릿, 모바일 환경에 맞는 반응형 디자인을 고민합니다.

### Design System
컬러, 타이포그래피, 컴포넌트, 버튼, 입력창 등  
재사용 가능한 UI 요소를 정리해 일관성 있는 화면을 만듭니다.

### 2. Moodget

**Moodget**은 사용자의 기분에 기반해 소비를 기록하고 돌아볼 수 있는  
소비 트래킹 앱 디자인 프로젝트입니다.

단순히 지출 금액만 기록하는 것이 아니라,  
사용자가 어떤 감정 상태에서 소비했는지 함께 기록함으로써  
소비 습관과 감정의 관계를 시각적으로 이해할 수 있도록 설계했습니다.

- Role: UX Research, App UI Design, Prototype
- Tools: Figma
- Focus: 감정 기반 소비 기록, 사용자 플로우, 모바일 앱 UI, 데이터 시각화

#### Key Features

- 기분 선택 후 소비 기록
- 날짜별 소비 내역 확인
- 감정별 소비 패턴 분석
- 직관적인 카드형 UI
- 소비 습관을 되돌아볼 수 있는 리포트 화면

### 3. 국립중앙박물관 Redesign

국립중앙박물관 웹사이트의 정보 구조와 시각적 경험을 개선하기 위한  
**웹 리디자인 프로젝트**입니다.

방문자가 전시, 관람 정보, 예약, 교육 프로그램 등 주요 정보를  
더 쉽고 빠르게 찾을 수 있도록 화면 구조를 재정리하고,  
한국적인 분위기와 현대적인 UI가 조화되도록 디자인 방향을 설정했습니다.

- Role: UI/UX Design, Web Redesign
- Tools: Figma
- Focus: 정보 구조 개선, 메인 화면 리디자인, 사용자 동선 개선, 시각적 아이덴티티

#### Redesign Goals

- 주요 관람 정보 접근성 개선
- 전시 콘텐츠의 시각적 강조
- 복잡한 메뉴 구조 정리
- 전통성과 현대성이 함께 느껴지는 UI 구성
- 사용자 중심의 예약 및 관람 동선 설계
